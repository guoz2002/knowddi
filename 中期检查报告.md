# 毕业设计（论文）中期检查报告

  ## 一、毕业设计（论文）研究进展情况
  截至目前，本项目已按照《项目实施计划》的安排，完成了前期环境搭建、数据准备、基准模型复现以及改进方案设计等工作，现已进入第三阶段的模型改进与消融实验实施过程，但该
  阶段尚未完成。当前研究进展如下：

  1. 已在本地与服务器环境中完成深度学习运行环境搭建，完成了 Python、PyTorch、PyTorch Geometric 及相关依赖库的配置，并解决了新硬件平台与原始代码环境不兼容的问题。
  2. 已完成 DrugBank 与 BioSNAP 两个数据集的导入、整理与预处理，同时接入外部生物医学知识图谱 Hetionet，为后续知识增强建模提供数据基础。
  3. 已成功跑通 KnowDDI 原始代码，完成基准模型训练、验证与测试流程，并梳理了模型保存、评估与解释性子图输出的基本逻辑。
  4. 已完成对原模型关键模块的初步分析，明确了后续改进的重点方向，包括自适应图结构解耦、知识子图去噪与补全机制以及轻量化结构设计。
  5. 已将改进模块初步接入主模型并开始开展实验验证，但当前实验效果尚未达到预期，模型性能仍有进一步优化空间，相关参数配置和结构设计仍在持续调整之中。

  ## 二、已取得的阶段性成果
  1. 成功完成了 KnowDDI 基准模型在当前实验环境中的复现，说明原始模型能够在现有环境下稳定运行，具备继续开展后续机制分析与轻量化实验的基础条件。

  2. 针对 DrugBank 数据集，当前已完成与原论文评估协议基本一致的基准复现，得到 Macro-F1 = 91.49、ACC = 92.86、Cohen's κ = 91.53。与原论文报告结果 Macro-F1 = 91.53 ±
  0.24、ACC = 93.17 ± 0.09、Cohen's κ = 91.89 ± 0.11 接近，说明本阶段在 DrugBank 上的复现实验总体有效，可作为后续机制解耦与轻量化实验的对比基线。[引用占位：KnowDDI
  原论文，后续定稿时替换为正式参考文献格式]

  3. 针对 TWOSIDES 数据集（代码目录名为 BioSNAP），当前已基于保存的最优模型复现得到 AUROC = 95.44、AUPRC = 94.11。上述两项结果与原论文报告结果 AUROC = 95.43 ±
  0.02、AUPRC = 94.14 ± 0.03 基本接近，说明模型在该多标签基准数据集上的复现实验同样总体有效。[引用占位：KnowDDI 原论文，后续定稿时替换为正式参考文献格式]

  4. 已完成去噪与补全模块的受控变体构建，并初步完成相关消融实验，能够对不同结构操作在不同任务设定下的边际贡献开展定量分析。

  5. 已完成轻量化方案的初步实现与统一评测，包括子图稀疏采样、多维特征压缩及协同优化方案，并获得了精度、推理时延与显存占用等阶段性结果，为后续适用边界与折中关系分析
  提供了实验基础。

  ## 三、存在的问题及解决思路
  1. 原始代码依赖较旧版本的 PyTorch 与 CUDA，而当前服务器硬件环境较新，前期在环境适配、依赖安装与兼容性修复方面耗费了较多时间。针对这一问题，已通过升级底层框架、调
  整依赖版本和修复部分兼容性代码的方式解决，后续环境已基本稳定。
  2. 当前虽然已完成两个基准数据集的基准复现，并完成改进模块的初步接入，但现阶段改进模型的效果尚不理想，尚未稳定优于基准模型。这表明当前改进方案在模块结构、超参数设
  置或训练策略上仍存在不足。
  4. 针对上述问题，后续将重点从以下几个方面继续优化：一是调整改进模块的结构设计，降低其对原始特征表达的破坏；二是系统调节关键超参数，提升训练稳定性；三是通过消融实
  验定位效果不理想的具体原因，判断哪些改进是真正有效的，哪些部分仍需回退或重构。

  ## 四、下一阶段的工作计划和研究内容
  1. 在现有改进模块已经完成初步接入的基础上，继续推进结构优化与参数调试，优先解决当前改进模型效果不理想的问题。
  2. 开展消融实验，分别验证去噪机制、补全机制以及后续轻量化改造对 DrugBank 核心指标（Macro-F1、ACC、Cohen's κ）和 BioSNAP 核心指标（AUROC、AUPRC、Accuracy）的实际贡
  献。
  3. 对比基准模型与改进模型在不同参数配置下的表现，系统分析性能波动来源，筛选出稳定有效的改进方案。
  4. 在后续实验中加入推理延迟、显存占用等效率指标，构建“精度-效率”对照结果，为边缘部署场景分析提供实验依据。
  5. 整理当前复现结果、原论文结果及后续改进实验结果，逐步形成统一的实验记录表、图表材料与论文正文草稿，为毕业论文撰写做好准备。
