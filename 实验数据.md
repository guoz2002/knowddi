## 3. 综合实验分析

从两个数据集的消融结果可以看出，去噪与补全模块对模型性能的影响具有明显的数据集相关性与指标差异性。首先，在 DrugBank 多分类任务上，仅去噪模型取得了最高的 `Macro-F1`，而联合模型在 `ACC` 和 `Cohen's κ` 上达到最优。这说明去噪机制更有利于提升类别均衡意义下的识别能力，而补全机制更偏向于增强整体预测一致性。换言之，去噪主要作用于“减少噪声传播”，补全主要作用于“增强主流结构模式”，两者在 DrugBank 上形成了较明显的性能权衡关系。

其次，在 TWOSIDES（代码目录名为 BioSNAP）多标签任务上，四组模型之间的差异明显缩小，且原始基线模型在 `AUROC` 和 `AUPRC` 上仍保持最优。这说明在该场景下，KnowDDI 原始结构已经能够较充分地建模有效关系，改进模块虽然没有带来额外的整体性能增益，但仍可用于分析去噪与补全的作用边界。结合 `biosnap_completion_only` 训练日志中的图结构统计可见，TWOSIDES 上补全模块的激活比例在 3 个随机种子下均保持较低水平，说明当前补全策略在该数据集上较为保守，实际引入的新边有限，因此其边际贡献也相对有限。这一现象进一步说明，不同数据集对结构补全机制的需求并不一致。

| 数据集 | 实验设置 | completion_activation_ratio |
| --- | --- | --- |
| TWOSIDES（代码目录名为 BioSNAP） | completion-only | 0.62% - 0.79%（3 seeds） |

综合 DrugBank 与 TWOSIDES（代码目录名为 BioSNAP）的实验结果，可以得到一个更稳健的结论：去噪与补全模块的价值不在于简单保证所有数据集、所有指标上都全面超过基线，而在于它们在不同任务设定下呈现出可区分、可量化的边际贡献模式。DrugBank 的结果表明，去噪更有助于优化类别均衡性能，补全更有助于提升整体一致性；TWOSIDES 的结果则表明，当数据本身的结构表达已较充分时，补全模块的增益会减弱，模型更接近于维持基线性能。因而，本研究的核心贡献在于对图结构学习中的“去噪”与“补全”功能进行了定量解耦，并揭示了两类结构操作在小样本 DDI 预测中的适用范围与性能权衡，而不仅仅是追求统一指标上的绝对最优。

## 4. 可直接用于论文的结论表述

消融实验表明，图结构学习中的去噪与补全机制具有差异化边际贡献。在 DrugBank 数据集上，仅去噪模型取得最高 `Macro-F1`，而联合模型在 `ACC` 与 `Cohen's κ` 上表现最佳，说明去噪更有利于提升类别均衡意义下的预测性能，补全更有利于增强整体预测一致性。在 TWOSIDES（代码目录名为 BioSNAP）数据集上，原始基线模型在 `AUROC` 与 `AUPRC` 上仍保持最优，说明当原始图结构已经能够充分表达有效关系时，额外的结构补全与去噪并不一定带来显著增益。上述结果说明，去噪与补全并非简单叠加关系，而是在不同任务场景中呈现出不同的边际贡献和适用性边界，从而为 KnowDDI 图结构学习机制的功能解耦与性能分析提供了实证依据。

# 后续待补：药物对案例解释

为支撑论文中关于模型可解释性的论述，后续需补充 2 到 4 个典型药物对案例，对预测结果对应的知识子图路径进行人工整理与药理学解释。建议每个案例至少包含以下内容：

1. 药物对名称与预测的 DDI 类型。
2. 模型关注的关键实体、关系边与主要推理路径。
3. 去噪或补全模块对该案例结果产生影响的具体表现。
4. 与已知药理机制、靶点关系或文献证据的一致性说明。

该部分主要用于补充“生物医学证据支持与路径解释”层面的论证，不作为当前模块边际贡献分析与轻量化结论的前提条件。

可优先补做的两个候选案例如下：

1. Atropine 与 Scopolamine
   预测相互作用类型：可能增加对方的肌病/横纹肌溶解相关活性（DDI type 71）。
   预测结果：baseline 模型预测正确，预测分数为 0.986014。
   当前已拿到的结构证据：存在 Atropine → Compound::DB00725、Atropine → Scopolamine 等高权重边，同时 Atropine 还与多个邻近药物节点保持较强连接。
   当前可写的解释表述：该案例中，模型并非仅依赖 Atropine 与 Scopolamine 之间的单一连接，而是综合利用了 Atropine 周围的局部高权重邻域信息，对“肌病/横纹肌溶解相关活性增强”这一相互作用类型作出判断。因其高权重边相对分散，后续更适合写成“局部邻域共同支持预测”的案例。

2. Reboxetine 与 Atomoxetine
   预测相互作用类型：可能增加对方的支气管收缩相关活性（DDI type 53）。
   预测结果：baseline 模型预测正确，预测分数为 0.999673。
   当前已拿到的结构证据：存在 Reboxetine → Compound::DB01146 → Atomoxetine 的连续高权重连接，其中 309 → 16879 与 16879 → 610 均为原始边，路径结构相对清晰。
   当前可写的解释表述：该案例中，模型主要依据 Reboxetine 与桥接化合物 Compound::DB01146 的连接，以及该桥接化合物与 Atomoxetine 的连接，形成了一条较清晰的知识路径，并据此支持“支气管收缩相关活性增强”这一相互作用类型的预测。该案例路径更完整，适合作为优先展示案例。
