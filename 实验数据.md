# KnowDDI 改进模型实验数据

## 1. DrugBank 消融实验结果

DrugBank 为多分类 DDI 预测任务，统一采用 `Macro-F1`、`ACC` 和 `Cohen's κ` 作为评价指标。

| Model | Macro-F1 | ACC | Cohen's κ |
| --- | ---: | ---: | ---: |
| Baseline | 91.49 | 92.86 | 91.53 |
| Denoise-only | **91.57** | 92.66 | 91.31 |
| Completion-only | 90.00 | 92.89 | 91.57 |
| Denoise+Completion | 90.78 | **92.99** | **91.68** |

## 2. BioSNAP 消融实验结果

BioSNAP 为多标签 DDI 预测任务，统一采用 `AUROC`、`AUPRC` 和 `AP@50` 作为评价指标。

| Model | AUROC | AUPRC | AP@50 |
| --- | ---: | ---: | ---: |
| Baseline | **95.44** | **94.11** | **89.68** |
| Denoise-only | 95.21 | 93.79 | 89.45 |
| Completion-only | 95.23 | 93.94 | 89.35 |
| Denoise+Completion | 95.21 | 93.90 | 89.31 |

## 3. 综合实验分析

从两个数据集的消融结果可以看出，去噪与补全模块对模型性能的影响具有明显的数据集相关性与指标差异性。首先，在 DrugBank 多分类任务上，仅去噪模型取得了最高的 `Macro-F1`，而联合模型在 `ACC` 和 `Cohen's κ` 上达到最优。这说明去噪机制更有利于提升类别均衡意义下的识别能力，而补全机制更偏向于增强整体预测一致性。换言之，去噪主要作用于“减少噪声传播”，补全主要作用于“增强主流结构模式”，两者在 DrugBank 上形成了较明显的性能权衡关系。

其次，在 BioSNAP 多标签任务上，四组模型之间的差异明显缩小，且原始基线模型在 `AUROC`、`AUPRC` 和 `AP@50` 上仍保持最优。这说明在 BioSNAP 场景下，KnowDDI 原始结构已经能够较充分地建模有效关系，改进模块虽然没有带来额外的整体性能增益，但仍可用于分析去噪与补全的作用边界。结合训练日志中的图结构统计可见，BioSNAP 上补全模块的激活比例始终较低，说明当前补全策略在该数据集上较为保守，实际引入的新边有限，因此其边际贡献也相对有限。这一现象进一步说明，不同数据集对结构补全机制的需求并不一致。

综合 DrugBank 与 BioSNAP 的实验结果，可以得到一个更稳健的结论：去噪与补全模块的价值不在于简单保证所有数据集、所有指标上都全面超过基线，而在于它们在不同任务设定下呈现出可区分、可量化的边际贡献模式。DrugBank 的结果表明，去噪更有助于优化类别均衡性能，补全更有助于提升整体一致性；BioSNAP 的结果则表明，当数据本身的结构表达已较充分时，补全模块的增益会减弱，模型更接近于维持基线性能。因而，本研究的核心贡献在于对图结构学习中的“去噪”与“补全”功能进行了定量解耦，并揭示了两类结构操作在小样本 DDI 预测中的适用范围与性能权衡，而不仅仅是追求统一指标上的绝对最优。

## 4. 可直接用于论文的结论表述

消融实验表明，图结构学习中的去噪与补全机制具有差异化边际贡献。在 DrugBank 数据集上，仅去噪模型取得最高 `Macro-F1`，而联合模型在 `ACC` 与 `Cohen's κ` 上表现最佳，说明去噪更有利于提升类别均衡意义下的预测性能，补全更有利于增强整体预测一致性。在 BioSNAP 数据集上，原始基线模型仍保持最优，说明当原始图结构已经能够充分表达有效关系时，额外的结构补全与去噪并不一定带来显著增益。上述结果说明，去噪与补全并非简单叠加关系，而是在不同任务场景中呈现出不同的边际贡献和适用性边界，从而为 KnowDDI 图结构学习机制的功能解耦与性能分析提供了实证依据。
